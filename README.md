# 📘 Software Education Volunteering (SW 교육 기부 활동)

Welcome to this repository 👋  
This repository contains educational materials created as part of the  
**2024 First-Half “Visiting SW Education Volunteering Program”** organized by  
**Code Club Korea**.  

The materials are designed for **elementary and middle school students**  
and focus on **teaching Python programming fundamentals** through hands-on lessons.


---

## 🧑‍🏫 활동 개요

- 활동명: 2024 상반기 코드클럽 찾아가는 SW 교육기부단
- 주관: 코드클럽 한국위원회
- 소속: 대림대학교 AI시스템학과
- 팀명: 코드스쿨
- 봉사자: 손승하
- 교육 대상: 초등학교 6학년 ~ 중학교 1·2·3학년
- 교육 과목: **파이썬(Python) 기초**

---

## 🎯 활동 목적

- 프로그래밍 경험이 없는 학생들을 대상으로  
  **파이썬 기초 개념을 쉽고 재미있게 전달**
- 실습 중심 수업을 통해  
  **논리적 사고력과 문제 해결 능력 향상**
- SW에 대한 흥미와 자신감 형성

---

## 📚 교육 자료 구성

본 저장소에는 주차별 수업 자료(PDF)가 정리되어 있습니다.

### 🗂 주차별 교육 내용

- **1–2주차**
  - 프로그래밍이란 무엇인가
  - 파이썬 소개 및 개발 환경(IDLE) 설치
  - 출력(print), 기본 문법 이해

- **3–4주차**
  - 데이터 종류(int, float, string, bool)
  - 변수 개념 및 값 저장
  - 간단한 연산 실습

- **5–6주차**
  - 불리언(bool) 자료형
  - 비교 연산자, 논리 연산자(and, or, not)
  - 조건 판단 연습 문제 및 실습

- **7–8주차**
  - for 반복문 개념 이해
  - range 함수 활용
  - 반복문 실습 및 응용 문제
  - 간단한 미니 프로그램 제작
